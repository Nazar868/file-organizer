# Дошка оголошень

Веб-застосунок "Дошка оголошень" на Express, EJS та Prisma (SQLite).

## Функціонал

- Головна сторінка зі списком оголошень (картки), пошуком по назві, сортуванням за датою та пагінацією (10 на сторінку)
- Сторінка перегляду оголошення з повною інформацією та кнопкою видалення
- Форма створення оголошення з клієнтською (HTML) та серверною валідацією
- Видалення оголошення через `fetch` з методом `DELETE`
- Сторінки 404 та 500 для обробки помилок

## Встановлення та запуск

```bash
npm install
npx prisma migrate dev --name init
npm start
```

Застосунок буде доступний на `http://localhost:3000`.

> **Примітка:** ця збірка була підготовлена в ізольованому середовищі без
> доступу до `binaries.prisma.sh`, тому команду `npx prisma migrate dev`
> потрібно виконати самостійно на своїй машині — вона завантажить потрібні
> Prisma-двигуни, згенерує клієнта та створить файл бази даних `dev.db` з
> таблицею `Announcement`.

Корисні команди під час розробки:

```bash
npm run dev            # запуск з автоперезавантаженням (node --watch)
npx prisma studio       # веб-інтерфейс для перегляду/редагування даних (localhost:5555)
npx prisma generate     # перегенерувати Prisma Client після зміни schema.prisma
```

## Технічні примітки

- Пошук по назві реалізовано **незалежно від регістру** вручну в JavaScript
  (а не через Prisma `contains` з `mode: "insensitive"`), оскільки SQLite не
  підтримує цей параметр і не робить `LIKE` нечутливим до регістру для
  кириличних символів.
- Сортування та пагінація без активного пошуку виконуються нативно через
  Prisma (`orderBy`, `skip`, `take`, `count`) для ефективності.
- Помилки в асинхронних маршрутах автоматично потрапляють в error handler
  завдяки Express 5 (без потреби у `try/catch` у кожному маршруті).
